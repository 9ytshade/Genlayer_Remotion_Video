# GenLayer Remotion Video System

A cinematic Remotion-based video system for GenLayer — the intelligent blockchain with AI-powered smart contracts, GenVM, and Optimistic Democracy.

## Overview

This project produces 20 high-quality explainer videos covering GenLayer's technology, use cases, and ecosystem:

- What is GenLayer?
- Why GenLayer Exists
- Intelligent Contracts & GenVM
- Optimistic Democracy & Equivalence Principle
- Intelligent Oracles & Security
- Use Cases (DeFi, DAOs, Identity)
- Developer Experience & Roadmap
- Partnerships & Interoperability

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run studio` | Launch Remotion Studio for preview and editing |
| `npm run render` | Render a single video |
| `npm run render:all` | Render all 20 videos |
| `npm run build` | Bundle for production |
| `npm run upgrade` | Upgrade Remotion dependencies |

## Tech Stack

- [Remotion](https://remotion.dev) 4.x
- React 18
- TypeScript
